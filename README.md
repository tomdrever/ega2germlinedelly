# ega2germlinedelly

Small [GWF]() workflow for running germline . Optionally 

## Requirements

- python3 >= 3.10.12
- [gwf](https://github.com/gwforg/gwf) >= 2.0.0
- [singularity](https://sylabs.io/singularity) >= 3.11.0
- [pyega2](https://github.com/EGA-archive/ega-download-client) >= 5.1.0

## Installation

```
git clone git@github.com:panprostate/panprostate_workflows.git
python3 -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
```
